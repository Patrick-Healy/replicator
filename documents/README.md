# Supporting Documents

This folder contains supporting materials required by DCAS Rules 10-12.

## Contents

Place the following documents here as applicable:

### Survey/Experiment Materials [DCAS Rule 10]
- `survey.pdf` - Survey instrument
- `experiment_instructions.pdf` - Experiment protocols
- `subject_selection.pdf` - Subject selection criteria

### Ethics Documentation [DCAS Rule 11]
- `ethics_approval.pdf` - IRB/ethics committee approval letter
- `consent_form.pdf` - Participant consent form (template)

### Pre-Registration [DCAS Rule 12]
- `preregistration.pdf` - Pre-registration document
- Link to registry: [AEA RCT Registry / OSF / EGAP URL]

## Notes

- These documents are NOT part of the code repository
- Share via cloud storage (Dropbox, Google Drive) with collaborators
- Include copies in the final replication package
